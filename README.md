### English:

### fmavm (File MaVM)
fmavm is a terminal program for working with MaVM files.

## Link to the MaVM format repository: [SoPepo32/MaVM](https://github.com/SoPepo32/mavm)

## Usage:
First, install Python 3 (Python 3.10 is recommended because that's what I used for testing).
Second, install the dependencies with:
```terminal
python3 -m pip install -r requirements.txt
```
from the code folder.

Third, run:
```terminal
python3 main.py
```
If you don't want to use it this way, you can use the precompiled version for Linux x86_64. If you don't have Linux x86_64, you can compile it yourself.

## Compilation Steps (Git must be installed):
```terminal
git clone https://github.com/SoPepo32/fmavm.git
```
```terminal
cd fmavm
```
If using Linux:
```terminal
bash compiler.sh
```
If using Windows:
Double-click compiler.bat
Then, if using Linux, go to the dist folder created inside the fmavm folder and run:
```terminal
./FMaVM -h
```
If using Windows, go to the dist folder created inside the fmavm folder and run:
```terminal
FMaVM.exe -h
```
